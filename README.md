# DNS Resolver Tool

A simple and efficient **DNS Resolver** tool written in Python. This project allows users to query domain names and retrieve their corresponding IP addresses. It also supports reverse DNS lookups and basic error handling.

## Features

- Resolve domain names to IPv4 and IPv6 addresses.
- Perform reverse DNS lookups (IP → Hostname).
- Handles invalid domain inputs gracefully.
- Lightweight and fast DNS querying.
- Can be extended for network security and penetration testing tasks.
